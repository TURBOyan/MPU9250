模拟I2C方式读取MPU9250的融合了磁力计的MPL库


移植步骤：
1、用git bash将文件clone进工程文件夹内
2、添加C文件和H文件路径
4、在需要用到MPU9250的文件内添加#include "MPU_ForUser.h"
3、修改MPU9250_Config.h配置文件，引脚，GPIO口操作等
4、查看MPU_ForUser.h调用初始化和读值函数

v1.0.0  实现了MPU9250的MPL库的读取，偏航角十分钟漂移不超过1度，但磁力计原始值还无法读取
v2.0.0  实现了磁力计原始值读取
v2.1.0  添加了中断接收FIFO模式，提高了稳定性，节省了时间
